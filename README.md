# 2021 Advent of Code

[Advent of Code (AOC)](http://adventofcode.com/2021) exercises in Zig

## How to Install Zig
```
% brew install zig --HEAD
# to get updated builds...
% brew reinstall zig
```

## How to Build a Day
```
% mkdir day01
% cd day01
% zig init-exe
% vim src/main.zig
...
% zig test src/main.zig
% zig build run
```
